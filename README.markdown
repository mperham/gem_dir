Gem Dir
========

Prints the root directory of the specified gem.
  
  gem dir rails
  
Installation:
------------

Just install like any normal gem:
  gem install mperham-gem_dir
  
You might need to add Github as a gem source:
  gem sources -a http://gems.github.com
  
For more help: 
  gem dir --help
